ESIA | Arquitectura del Enrutador Híbrido

Animación del ciclo de vida de una consulta en el embudo de 4 niveles. Observa cómo el algoritmo destila la complejidad léxica y semántica antes de ejecutar la Recuperación RAG.

Nivel I: Gravedad de Interfaz
Latencia: Pasivo (Instantáneo)

El sistema captura el contexto visual del usuario. Identifica en qué dashboard de la web se encuentra operando y aplica un sesgo matemático inicial.

  • Asigna una constante de +0.05 puntos al agente correspondiente.
  • Funciona como un "Bono UI" que se almacenará en la memoria para el Nivel III.
// Escenario: El alumno navega en la pestaña de Bienestar Estudiantil.
Variables de estado: agente_ui = "bienestar"
► Bono Gravitacional Acumulado: Bienestar (+0.05)
Nivel II: Motor Heurístico (RegEx)
Latencia: ~0.0001s

El texto entra a una matriz de Expresiones Regulares de altísima velocidad. Compara la pregunta cruda con diccionarios predefinidos de áreas específicas.

  • Si hay coincidencia directa y única, se produce un Knockout Técnico y salta directo al RAG.
  • Si hay choque de palabras de distintas áreas, aplica un bono plano (+0.10) y delega al Nivel III.
// Caso Simple: "¿A qué hora sirven los menús hoy?" -> WIN: BIENESTAR

// Caso Complejo: "¿Qué pasa si me retiro del curso?"
⚠️ Ambigüedad Léxica: ['gestion_admin', 'tutoria'] -> Delegando a Nivel 3...
Nivel III: Enrutador Semántico Ponderado
Latencia: ~0.04s

El núcleo matemático. Transforma la frase en vectores (all-MiniLM-L6-v2) y calcula la Similitud del Coseno. Le suma los bonos del Nivel 1 y 2, y evalúa la Matriz de Confianza Bidimensional.

  • Out-of-Domain: Si MaxScore < 0.25 -> Bloqueo absoluto.
  • Botones UX: Si MaxScore >= 0.55 Y Diferencia < 0.08 -> Lanza opciones al usuario.
  • Válvula de Escape: Si 0.25 <= MaxScore < 0.40 -> Falla de confianza, delega al LLM.
// Simulación de Choque Vectorial:
Similitud Base: Tutoría(0.53) | Soporte(0.48)
+ Bono Nivel 2: Tutoría(+0.10) | Soporte(+0.10)
Puntaje Final: Tutoría(0.63) vs Soporte(0.58) | Diferencia: 0.05
► Acción: Desplegar interfaz de botones para Desambiguación.
Nivel IV: Zero-Shot LLM (Contingencia)
Latencia: ~11.00s

El último escudo de contención. Si el texto es demasiado abstracto, poético o complejo para las matemáticas tradicionales, el servidor despierta a Llama 3.1 para que razone y clasifique el texto pura y exclusivamente por inteligencia artificial generativa.

// Consulta Abstracta:
"Siento que el peso de mis responsabilidades académicas está quebrando mi mente, busco un remanso de paz para evitar el colapso."

// Análisis: Score Max (0.35) < Umbral Confianza (0.40)
► Llama 3.1 Zero-Shot Classification: BIENESTAR_ESTUDIANTIL
